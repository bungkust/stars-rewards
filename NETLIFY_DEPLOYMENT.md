# Netlify Deployment Guide - Stars Rewards

## 📋 Overview
Panduan lengkap untuk deploy Privacy Policy dan Terms of Use ke domain `starhabit.web.id` menggunakan Netlify.

## 🚀 Quick Start

### 1. **Install Netlify CLI**
```bash
npm install -g netlify-cli
# atau
yarn global add netlify-cli
```

### 2. **Login ke Netlify**
```bash
netlify login
```

### 3. **Deploy Site**
```bash
# Dari root project
npm run deploy:netlify
```

### 4. **Setup Custom Domain**
1. Buka [Netlify Dashboard](https://app.netlify.com)
2. Pilih site yang baru di-deploy
3. Go to **Domain management**
4. Add custom domain: `starhabit.web.id`
5. Configure DNS sesuai instruksi Netlify

## 📁 Project Structure

```
docs/
├── index.html          # Homepage
├── privacy.html        # Privacy Policy
├── terms.html          # Terms & Conditions
└── _redirects          # URL redirects (auto-generated by netlify.toml)
```

---

## ⚙️ Configuration Files

### **netlify.toml**
```toml
[build]
  publish = "docs"
  command = "echo 'Static site - no build needed'"

[[redirects]]
  from = "/privacy"
  to = "/privacy.html"
  status = 200

[[redirects]]
  from = "/terms"
  to = "/terms.html"
  status = 200

# SPA fallback
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200
```

### **Package.json Scripts**
```json
{
  "scripts": {
    "deploy:netlify": "npm run build && netlify deploy --dir=docs --prod"
  }
}
```

## 🛠️ Manual Deployment (Alternative)

### **Via Netlify Dashboard**
1. **Drag & Drop:**
   - Zip folder `docs/`
   - Upload ke [Netlify Drop](https://app.netlify.com/drop)

2. **Git Integration:**
   - Connect GitHub repo
   - Set build command: `echo 'Static site'`
   - Set publish directory: `docs`

### **Via Netlify CLI**
```bash
# Initialize site
netlify init

# Deploy to production
netlify deploy --dir=docs --prod

# Deploy to draft (preview)
netlify deploy --dir=docs
```

## 🌐 Custom Domain Setup

### **DNS Configuration**
1. **Login ke Domain Registrar** (starhabit.web.id)
2. **Add CNAME record:**
   ```
   Name: @
   Type: CNAME
   Value: [your-netlify-site].netlify.app
   ```

3. **Add Netlify nameservers** (recommended):
   ```
   ns1.netlify.com
   ns2.netlify.com
   ns3.netlify.com
   ns4.netlify.com
   ```

### **SSL Certificate**
- Netlify otomatis provide SSL certificate
- HTTPS akan aktif secara otomatis

## 🚨 Troubleshooting

### **404 Errors**
```bash
# Check if files exist in docs/
ls -la docs/

# Check netlify.toml redirects
cat netlify.toml

# Test locally
cd docs && python -m http.server 8000
# Visit http://localhost:8000/privacy
```

### **Domain Issues**
```bash
# Check DNS propagation
nslookup starhabit.web.id

# Check Netlify domain settings
# Go to Site Settings → Domain management
```

### **SSL Issues**
- Netlify otomatis handle SSL
- Tunggu 24 jam jika belum aktif
- Check site settings di Netlify dashboard
